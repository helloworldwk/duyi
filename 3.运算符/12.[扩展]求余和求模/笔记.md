# 求余和求模

%：求余

x % y

1. 求余 x rem y：x - n * y，n表示商取整（**直接去掉小数，向0取整**）

```
  x = y, y = 3;

  n = x / y = 2.333333 ≈ 2;
  x rem y = x - n * y = 7 - 2 * 3 = 1;
```

```
  x = y, y = -3;

  n = x / y = -2.333333 ≈ -2;
  x rem y = x - n * y = 7 - (-2) * -3 = 1;
```

**求余的符号与被除数相同**

2. 求模 x mod y：x - n * y，n表示商取整（**向下取整**）

```
  x = 7, y = -3;

  n = x / y = 7 / -3 = -2.3333333 ≈ -3;
  x mod y = x - n * y = 7 - (-3) * -3 = 7 - 9 = -2;
```

```
  x = -7, y = 3;

  n = x / y = -7 / 3 = -2.3333333 ≈ -3;
  x mod y = x - n * y = -7 - 3 * -3 = -7 + 9 = 2;
```

**求模的符号与除数相同**
